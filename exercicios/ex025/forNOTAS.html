Todos os Elementos de formulários aprendidos no capitulos 24

Inputs

Input funções
text Permite inserir texto comum (como nome, endereço, etc.).
password Similar ao text, mas oculta o conteúdo inserido, substituindo-o por pontos ou asteriscos.
number Permite inserir apenas valores numéricos, e normalmente exibe setas para aumentar ou diminuir o valor.
data Exibe um seletor de data para escolher um dia, mês e ano.
month Permite selecionar apenas o mês e o ano (sem um dia específico).
color Exibe um seletor de cor para escolher um valor hexadecimal de cor.
range Exibe um controle deslizante para selecionar um valor dentro de um intervalo.
file Permite ao usuário selecionar um arquivo do sistema para upload.
submit Envia o formulário quando clicado.
reset Redefine todos os campos do formulário para seus valores padrão.

Novas Tags & Atributos

Atributos Funções
Form é usada para criar um formulário HTML, que pode conter campos de entrada (como texto, botões, seleções, etc.) e ser enviado para um servidor..
label é usada para associar um rótulo a um campo de formulário. Isso melhora a acessibilidade, permitindo que os usuários cliquem no rótulo para focar no campo correspondente.
fieldset é usada para agrupar visualmente elementos relacionados dentro de um formulário. Ela cria uma borda ao redor do grupo de campos.
legend é usada dentro de um fieldset para fornecer um título ou legenda para o grupo de campos.
Atributos Funções
required Indica que o campo é obrigatório.
minlenght Define o número mínimo de caracteres que o usuário deve inserir.
maxlenght Define o número máximo de caracteres permitidos no campo.
size Define a largura do campo de texto, em número de caracteres visíveis.
placeholder Exibe um texto de exemplo dentro do campo, que desaparece quando o usuário começa a digitar.
autocomplete Indica se o navegador deve preencher automaticamente o campo com dados previamente armazenados.
step Define o intervalo entre os valores válidos para campos numéricos ou de data.
min/max Define o valor mínimo e máximo permitidos para campos numéricos ou de data.
value Define um valor inicial padrão para o campo.
pattern Define uma expressão regular para validação do campo.
